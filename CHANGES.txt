 2026-05-15  Global Admin, Webhooks & Email Notifications  (NOT YET COMMITTED)
────────────────────────────────────────────────────────────────────────────────

  GLOBAL ADMIN SUPPORT
  - Added isGlobalAdmin() helper in client/src/utils/access.js
      Logic: isSuperAdmin OR (role=admin AND no track assigned)
  - App.jsx: SuperAdminRoute now uses isGlobalAdmin (not just super admin)
  - Layout.jsx: "User Management" nav item shown to all global admins
  - UserManagement.jsx: save guard uses isGlobalAdmin
  - deps.py: added require_global_admin dependency on backend
  - users.py: all routes switched from require_super_admin to require_global_admin
  - pocs.py: enforce_admin_track_access() now passes global admins through freely

  POWER AUTOMATE WEBHOOK — PUBLISHED EMAIL
  - Fires when a contribution is published (via Publish button or edit + publish)
  - Sends to: all users EXCEPT the publisher
  - Added POWER_AUTOMATE_PUBLISHED_WEBHOOK_URL to config.py and .env
  - queue_published_notifications() function added in pocs.py
  - Wired into POST /publish endpoint
  - Wired into PUT /pocs/:id endpoint when status transitions to "published"

  POWER AUTOMATE WEBHOOK — USER APPROVED EMAIL
  - Fires when a user is approved as a contributor
  - Sends to: the single user who was just approved
  - Added POWER_AUTOMATE_USER_APPROVED_WEBHOOK_URL to config.py and .env
  - queue_user_approved_notification() function added in pocs.py
  - Wired into POST /approve-user endpoint
  - Wired into POST /add-contributor endpoint (directly-added users also notified)

  POWER AUTOMATE WEBHOOK — FINISHED EMAIL
  - Fires when a contribution is marked as finished
  - Sends to: all approved contributors + anyone who received credits
  - Added POWER_AUTOMATE_FINISHED_WEBHOOK_URL to config.py and .env
  - queue_finished_notifications() function added in pocs.py
  - Wired into POST /finish endpoint

  JSON PAYLOAD (same structure for all 4 webhooks)
  - eventType, triggeredAt, poc { id, title, track, status, link }
  - triggeredBy { id, name, email }
  - recipients [ { id, name, email } ... ]
  - Finished webhook also includes: poc.impact, poc.creditsAwarded
  - Live webhook also includes: poc.liveAt

  HTML EMAIL TEMPLATES (designed for Power Automate)
  - All 4 emails given professional HTML templates with colour-coded headers
  - Live = green, Published = indigo, Approved = amber, Finished = dark slate
  - Table-based layout with inline CSS for Outlook compatibility
